# magicLoad.js

A dynamic loading .js. Load your elements when they are really can be seen.

一个用于元素动态加载的.js。使特定元素在**确实可见**时被加载。

## Usage
Notice: Need JQuery
```
<script language="JavaScript" src="js/magicLoad.js"></script>
<script language="JavaScript" >
    magicLoad($("#main"),{
        rule_list:[6]
    });
</script>
```
